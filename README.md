## UniPro Kite Admin Panel

Laravel template for admin panel, base template using [Unipro Kite - 001 Blue Design](https://preview.themeforest.net/item/unipro-bootstrap-5-html-admin-template/full_screen_preview/32619252?_ga=2.266130504.691362346.1699409565-1477051943.1699409565) I got free from Envato monthly suprise.

- [Laravel v10.28.0](https://laravel.com/docs) (PHP v8.1.16)
- [Bootstrap 5](https://getbootstrap.com/)
- [Jquery v3.6.0](https://jquery.com/)

## How to install

<blockquote>
    <ol>
        <li>Clone this repo with <code>git clone https://github.com/asepnurjaman-en/panel-unipro-kite-laravel-bs4.git</code></li>
        <li>Run <code>cd panel-unipro-kite-laravel-bs4</code></li>
        <li>Run <code>composer install</code></li>
        <li>Run <code>cp .env.example .env</code></li>
        <li>Run <code>php artisan key:generate</code></li>
        <li>Run <code>php artisan migrate</code></li>
        <li>Run <code>php artisan serve</code></li>
        <li>Open http://localhost:8000</li>
    </ol>
</blockquote>
